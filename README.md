### Getting Started

You can install Create Expo Kit in various ways:

NPM:
```sh
$ npm install -g create-expo-kit
```

Cargo:
```sh
$ cargo install create-expo-kit
```

Homebrew:
```sh
$ brew install create-expo-kit
```
